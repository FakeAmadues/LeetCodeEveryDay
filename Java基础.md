### 3、基本数据类型

```java
1.整数类型：byte(1)|2^7-1|,short(2)|2^15-1|,
		  int(4)|2^31-1|,long(8)|2^63-1|(声明long时要加L or l)
2.浮点小数类型：由符号位+指数为+尾数位组成,浮点数相等判断应该用Math.abs
              float[4](声明float时要加F or f),double[8]
3.字符型：char[2]字符常量常用单引号'',java允许使用转义符 ""代表字符串
         存储：'a'=>码值97=>二进制=>存储
         读取：二进制=>码值97=>'a'=>显示
4.布尔类型：boolean[1] java不能使用0 or 1来表示boolean值
5.自动类型转换：char->int->long->float->double
              byte->short->int->long->float->double
6.强制类型转换：将容量大的转换成容量小的，可能会造成精度降低或溢出
              char类型可以保存int的常量值不能保存变量值
              强转符号只对最近的操作数有效可以使用小括号提升优先级
```

## 第三章

### 1、运算符

```java
1.算术运算符：+ - * / %(a-a/b*b) 
    		++(前++先自增后赋值 后++先赋值后自增) --(与++同理)
2.关系运算符：(boolean)== != < > <= >= instanceof(检查是否是类对象)
3.逻辑运算符：&(逻辑与) &&(短路与) | || ! ^(异或，不同是为true)
    		逻辑与：不管第一个条件是否为false，第二个条件都要判断
    		短路与：如果第一个条件为false，则第二个条件不会判断，效率更高
    		逻辑或：不管第一个条件是否为ture，第二个条件都要判断
    		短路或：如果第一个条件为true，则第二个条件不会判断，效率更高
4.赋值运算符：= += -= *= /= %= 从右往左
    		符合运算符会进行类型转换：byte b=2;b+=2(b=(byte)(b+2));
5.三元运算符：条件表达式?表达式1:表达式2;条件表达式为真->表达式1，三元运算符为一个整体
6.运算符优先级：a.(),{}等 b.单目运行++ c.算术运算符 d.位移运算符
    		  e.比较运算符 f.逻辑运算符 g.三元运算符
```

### 3、进制

```java
1.二进制(0b)、十进制、八进制(0)、十六进制(0X)
2.二进制转十进制:0b1011  1*2^0+1*2^1+0*2^2+1*2^3=11
3.八进制转十进制：0234	 4*8^0+3*8^1+2*8^2=156
4.十六进制转十进制：0x23A 10*16^0+3*16^1+2*16^2=570
5.十进制转其他进制：将该数不断除以进制数，直到商为0，然后将每步得到的余数倒过来。注意二进制要进行相应的补位
6.二进制转八进制、十六进制：每三位转换一个数、每四位转换一个数
7.十六进制、八进制转二进制：与以上规则相反
```

### 4、原码、反码、补码

```java
1.二进制的最高位是符号位：0表示正数、1表示负数
2.正数的原码、反码、补码都一样(三码合一)
3.负数的反码=它的原码符号位不变，其它按位取反    
4.负数的补码=它的反码+1，负数的反码=负数补码-1    
5.0的反码、补码都是0    
6.java没有无符号数，换言之，java中的数都是有符号的
7.计算机运算的时候，都是以补码的方式来运算的    
8.当我们看运算结果的时候，要看他的原码。    
```
